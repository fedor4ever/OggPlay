Installation instructions
-------------------------

1. Install OggPlayS60V1_1_72.sis


New users
---------

Once installed you can start OggPlay and search for audio files (Select "Find new files" from the Options menu)


Package contents
----------------

OggPlay installation file:
OggPlayS60_1_72.sis

Recognizer installation file:
Recognizer.S60.sis
Install this if you want other applications (e.g. FExplorer) to be able to launch OggPlay
Note that the recognizer will be installed to the memory card.
